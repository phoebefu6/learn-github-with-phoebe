# learn-github-with-phoebe

GitHub for everyone - no code required. 6 browser-only sessions, zero code, zero command line. You build a real team handbook as the running artifact: created in Session 2, planned in Session 3, collaborated on in Session 4, live on the internet by Session 5, health-checked like a pro in Session 6.

## Sessions

| # | Title | What you learn | Exercise |
|---|-------|----------------|----------|
| 1 | Meet GitHub | What GitHub is, account-org-repo map, profiles and stars | Where-does-this-live sorter |
| 2 | Your first repo + Markdown | Create a repo, README front door, commits, everyday Markdown | Live markdown playground |
| 3 | Issues + Projects | Issues as the universal todo, labels, milestones, kanban boards | Triage-this-issue sorter |
| 4 | Branches + PRs without code | Safe copies, proposals, line-level review, merge | PR journey sorter |
| 5 | Publish with GitHub Pages | Repo to live website via Settings only | Go-live checklist |
| 6 | Speak fluent GitHub | Green checks and Actions literacy, the 90-second repo health read, the dev phrasebook | Repo health scorecard + phrasebook sorter |

## Official GitHub Skills mapping

- introduction-to-github - S1, S2, S4 (full working content)
- communicate-using-markdown - S2 (full working content)
- review-pull-requests - S4 (review flow via web UI)
- github-pages - S5 (settings-only path)
- hello-github-actions - S6 (literacy level: read + install, not write)
- introduction-to-repository-management - S3, S6 (community-health basics)

## Run locally

```
python3 -m http.server
```

Then open http://localhost:8000
