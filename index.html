✓ What this course is

Browser-only GitHub fluency: repositories, Markdown, issues and boards, pull requests and reviews, publishing with Pages, and repo literacy - reading any project's health and speaking the vocabulary. Everything a non-tech professional needs to work where modern teams work.

✗ What it is not

No git CLI, no code, no merge-conflict surgery, no Actions authoring, no Copilot - those live in the tech track, now live. Boundaries kept on purpose so the browser toolkit actually sticks.
